Decathlon API

https://decathlon-api.onrender.com/api/decathlonEvent

Copy/Paste the url above into the address bar and replace the word "decathlonEvent" in the above url with one of the events below

  1. 100 Meter Dash
  2. Long Jump
  3. Shot Put
  4. High Jump
  5. 400 Meter Dash
  6. 110 Meter Hurdles
  7. Discus
  8. Pole Vault
  9. Javelin
  10. 1500 Meter Run

BONUS: try /api/decathlon to see data for the decathlon.
